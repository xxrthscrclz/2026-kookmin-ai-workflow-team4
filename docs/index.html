핵심 기능
담록이 제공하는 세 가지 메인 기능
📝
핵심
AI 회의록 생성
회의 내용을 입력하면 AI가 안건·논의 요약·결정사항으로 구조화된 회의록을 만들어 드립니다.
✅
핵심
액션 아이템 트래킹
회의에서 나온 할 일을 담당자·기한과 함께 추적합니다. todo / done 보드로 팀 실행력을 높입니다.
🔍
핵심
과거 회의 검색
키워드로 지난 회의록을 빠르게 찾습니다. 이미 결정된 내용을 다시 논의하지 않도록 돕습니다.
이용 방법
4단계로 담록을 사용해 보세요
01
회의 내용 입력
회의록 생성 페이지에 텍스트·제목·참석자를 입력하거나 파일을 업로드합니다.
02
AI 구조화
AI가 안건, 논의 요약, 결정사항, 액션 아이템을 자동으로 추출합니다.
03
액션 추적
트래커 보드에서 담당자별 할 일 상태를 관리합니다.
04
회의 검색
필요할 때 과거 회의를 검색해 팀의 지식을 다시 꺼내 씁니다.
